Q1. What is jQuery?

jQuery is a JavaScript library designed to simplify HTML DOM tree traversal and manipulation, as well as event handling, CSS animations, and Ajax.

Q2. What is an API?

An API, or Application Programming Interface, is a set of rules and definitions that allow software applications to communicate with each other.

Q3. What is CSS?

CSS, or Cascading Style Sheets, is a stylesheet language used to describe the presentation of a document written in HTML or XML.
